VOT Travel – EV Charging Station Finder App ⚡

📌 Overview:

VOT Travel is a mobile application designed to help electric vehicle users easily locate nearby charging stations, check real-time availability, and plan their trips efficiently.

❗ Problem:
EV users often struggle with:
- Finding nearby charging stations
- Checking real-time charger availability
- Understanding charger types and compatibility
- Planning long-distance trips confidently

💡 Solution:
Designed a smart EV charging finder app that:
- Displays nearby charging stations on a map
- Shows real-time availability and charger details
- Filters stations by connector type and speed
- Helps users plan trips with charging stops

🛠 Tools Used:
- Figma
- FigJam
- Miro

👤 My Role:
- UX Research
- User Persona Creation
- Wireframing
- UI Design
- Prototyping

🎯 Outcome:

Delivered a user-friendly EV charging solution that reduces range anxiety and improves trip planning confidence for electric vehicle users.
